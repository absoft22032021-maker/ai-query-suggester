AI Query Suggester — Privacy Policy

Last updated: 2025‑09‑28

Overview

AI Query Suggester is a Chrome extension that converts short user inputs into Jira JQL, Stack Overflow search strings, and RingCentral bug reports.

Data We Collect

We do not collect, store, or transmit personal data to our own servers. We do not operate any backend.

The extension stores only minimal settings via Chrome storage:

  • chrome.storage.sync: provider, model, API key, custom model IDs.
  • chrome.storage.local: last selected text and the active UI mode.

How Your Data Is Used

When you click Generate, your input (and only your input) is sent to the chosen model provider (OpenAI or Google Gemini) to produce the requested output. No other parties receive the data.

Permissions

  • storage — save extension settings and small UI state.
  • tabs — open Jira Issue Navigator or Stack Overflow results in a new tab at your request.
  • contextMenus — provide a single menu item to use the selected text.
  • activeTab — access tab metadata for user-initiated actions (no page scraping).

No content scripts are injected. No remote code, eval, or runtime-fetched scripts.

Sharing & Selling

We do not sell user data. We do not share user data except with the selected model provider solely to provide the requested feature.

Security

API keys are stored via Chrome’s extension storage. Revoke keys if compromised; we recommend keys with usage limits.

Children

This extension is not directed at children under 13 and does not knowingly collect information from them.

Changes

We may update this policy and will change the date at the top of this page.

Contact

For privacy questions, please open an issue on the repository or contact the developer.
